# game-suite

Maths game suite MVP with a graph-isomorphism game.

## App location

The web app lives in `/home/runner/work/game-suite/game-suite/web`.

## Run locally

```bash
cd /home/runner/work/game-suite/game-suite/web
npm install
npm run dev
```

## Supabase setup

Create `/home/runner/work/game-suite/game-suite/web/.env`:

```bash
VITE_SUPABASE_URL=your-project-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```

Create this table in Supabase SQL editor:

```sql
create table if not exists public.scores (
  id bigint generated by default as identity primary key,
  user_id uuid not null references auth.users(id) on delete cascade,
  display_name text,
  score integer not null check (score >= 0),
  created_at timestamptz not null default now()
);

alter table public.scores enable row level security;

create policy "scores are readable by everyone"
on public.scores for select
using (true);

create policy "users can insert own score"
on public.scores for insert
with check (auth.uid() = user_id);
```

With env vars configured, users can sign up/log in, play 10 rounds, save final scores, and view a top-10 leaderboard.
